- Buat folder

- Initialisasi utuk buat dir .git
git init

- Mengindex nama file dan folder yang akan diupload
git add .

- Kometar untuk upload ke git
git commit -m "first commit"

- Buat repositori dan copy link dan paste ke
git remote add origin link_git
example  : 
git remote add origin https://github.com/maspamz/MateriBelajarGithub.git

- Proses send / upload file ke branch/versi dokumentasi
git push -u origin master

- Untuk clone repository
git clone link_github
example :
git clone https://github.com/maspamz/MateriBelajarGithub.git

- Proses syncoronisasi/ tarik data dari repository
git pull
